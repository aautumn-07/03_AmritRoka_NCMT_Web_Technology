CHAPTER 1: AN OVERVIEW OF THE INTERNET AND THE WEB
Group A: Short Questions

1. Define the term “Protocol” in the context of the Internet.

=> A protocol is a set of rules used for communication between computers on the Internet. It controls how data is sent, received, and understood so that devices can communicate correctly.

2. What is the primary difference between a Web Browser and a Search Engine?

=> A web browser is software used to open and view websites, while a search engine is an online service that helps users find information and websites on the Internet.

3. Define the World Wide Web (WWW).

=> The World Wide Web is a collection of web pages connected through hyperlinks and accessed using the Internet. It allows users to view text, images, videos, and other content.

4. Distinguish between a static web page and a dynamic web page.

=> A static web page shows the same content to all users, while a dynamic web page changes its content based on user interaction or data from a database.

Group B: Long Questions

5. Explain the client–server architecture of the web with a neat diagram.

=> Client–server architecture is a system where the client (web browser) requests information and the server provides it. When a user enters a website address, the browser sends a request to the server. The server processes the request and sends the required web page back to the client. This system allows many users to access the same website efficiently.

6. Describe the functions of Email, FTP, and VoIP.

=> Email is used to send and receive electronic messages, FTP is used to transfer files between computers over a network, and VoIP allows users to make voice calls using an Internet connection instead of traditional phone lines.

7. “The Internet is a network of networks.” Explain this statement and packet switching.

=> The Internet connects many smaller networks around the world to form a global network. Data is divided into small packets before transmission. Each packet travels through different paths and is reassembled at the destination, making communication fast and reliable.

Group C: Scenario-Based Questions

8. A startup wants a low-cost website showing the same content to all users. Should it use a dynamic website?

=> I do not agree with using a dynamic website because the company only needs to show the same information to all visitors. A static website is cheaper, easy to maintain, and does not require databases or complex programming, making it the best option.

9. A website opens with an IP address but not with a domain name. Explain the issue.

=> This problem occurs due to a failure in the Domain Name System (DNS). DNS converts domain names into IP addresses. Since the site works with an IP address, it shows that the DNS server is not working properly.

10. A company wants to share private data with selected suppliers. Which network should be used and why?

=> The company should use an Extranet because it allows controlled access to specific external users. It is more secure than the Internet and suitable for sharing sensitive data with trusted suppliers.
