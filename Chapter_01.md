CHAPTER 1: AN OVERVIEW OF THE INTERNET AND THE WEB
Group A: Short Questions

1. Define the term “Protocol” in the context of the Internet.
=> A protocol is a set of rules used for communication between computers on the Internet.
=> It decides how data is sent, received, and understood so that devices can communicate properly.

2. What is the primary difference between a Web Browser and a Search Engine?
=> A web browser is software used to open and view websites on the Internet.
=> A search engine is an online service that helps users find information and websites.

3. Define the World Wide Web (WWW).
=> The World Wide Web is a collection of web pages connected through links.
=> It allows users to access text, images, videos, and other content using the Internet.

4. Distinguish between a static web page and a dynamic web page.
=> A static web page shows the same content to all users and does not change automatically.
=> A dynamic web page changes content based on user input or data from a database.

Group B: Long Questions

5. Explain the client–server architecture of the web with a neat diagram.
=> Client–server architecture is a system where the client requests data and the server provides it.
=> The client is usually a web browser used by the user.
=> The server stores website files and processes requests.
=> When a user enters a website address, the browser sends a request to the server.
=> The server processes the request and sends the web page back to the browser.
=> This system helps manage data efficiently and allows many users to access the same website.

6. Describe the functions of Email, FTP, and VoIP.
=> Email is used to send and receive electronic messages over the Internet.
=> FTP is used to transfer files from one computer to another through a network.
=> VoIP allows users to make voice calls using an Internet connection instead of telephone lines.

7. “The Internet is a network of networks.” Explain this statement and packet switching.
=> The Internet connects many smaller networks around the world.
=> These networks work together to share information.
=> Data is broken into small packets before sending.
=> Each packet travels through different paths on the network.
=> At the destination, packets are joined together.
=> This method makes communication faster and more reliable.

Group C: Scenario-Based Questions

8. A startup wants a low-cost website showing same content to all users. Should it use a dynamic website?
=> I do not agree with using a dynamic website in this case.
=> The company only needs to show the same information to all visitors.
=> A static website is cheaper and easy to maintain.
=> It does not need a database or server-side programming.
=> Dynamic websites increase cost and complexity.
=> Therefore, a static website is the better choice.

9. Website opens with IP address but not with domain name. Explain the issue.
=> The problem is related to the Domain Name System (DNS).
=> DNS converts website names into IP addresses.
=> The browser could not translate the domain name.
=> When the IP address is used, DNS is skipped.
=> This means the DNS server is not working properly.
=> The failed technology here is DNS.

10. A company wants to share private data with selected suppliers. Which network should be used and why?
=> The company should use an Extranet.
=> An Extranet allows limited access to external users.
=> Only authorized suppliers can view the data.
=> It is more secure than the Internet.
=> An Intranet is only for internal staff.
=> Extranet is best for controlled data sharing.
